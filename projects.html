Projects

A little list of all the major personal projects I've worked on over the years.

Leadlight

Git

A DSP library written in Rust.

Algorithms are based off the Airwindows library, by Chris Johnson. Emphasis was placed on performance and clear documentation, so that other developers could use it as a reference implementation for other personal projects.

Coralia

Git

A local music player, written in Typescript and Rust.

The front-end uses a Typescript/React/TailwindCSS stack. It focuses on efficient rendering of large lists and many images, and adhering carefully to responsive design. The back-end is written in Rust, with a focus on efficient decoding of local audio files, supporting true gapless playback.

Terminal Typer

Git

A TUI Monkeytype clone, written in Rust.

FreeBSD Rootkit

Git

A FreeBSD root-kit, written in C. Researched and developed as part of a university security course.
